提供された情報を整理し、コーディングエージェントが履修判定ロジックやデータベース構築にそのまま利用できる構造化データとしてまとめました。

この資料は、直前の回答で解消した「必修単位の内訳」「コース別科目の詳細」「開講期」の曖昧さを排除した**実装用リファレンスデータ**です。

曜日時限の情報は補助資料として保持しますが、このプロジェクトでは実際の時間割作成は行わず、曜日衝突判定にも使いません。

## 0. 参照優先順位メモ
開講期や配当の判断で資料間に差や不足がある場合は、次の優先順位で補完する。

1. `カリキュラム（情報）.pdf`
2. `R8履修の手引き.pdf`
3. `2026前期情報メディア学科-1.pdf`
4. `2024後期情報メディア学科時間割４年ver4.pdf`

時間割PDFは、実際の開講実績を補うために使う。ただし年度差があるため、単独では恒久的な開講保証とはみなさない。

## 0.5. `R8履修の手引き.pdf` の利用範囲
`R8履修の手引き.pdf` は全学科を含むため、目次を基準に情報メディア学科に関係する章だけを利用する。

- 1. 教育課程
- 2. 単位制度（情報メディア学科）
- 3. 卒業・進級要件（情報メディア学科）
- 4. 履修（情報メディア学科）
- 情報メディア学部 情報メディア学科 教職課程の履修について

実際のPDF内部ページは冊子の通しページとずれるため、参照時は次の対応を使う。

| 冊子上の見出し | 冊子ページ | PDFページ |
| :--- | :---: | :---: |
| 教育課程 | 78 | 42 |
| 卒業・進級要件 | 91 | 55 |
| 履修 | 92-93 | 56-57 |
| 教職課程 | 110 | 74 |

このPDFからは、卒業要件文言、進級条件、履修登録上限、教職課程の基礎資格などを補助根拠として採用する。

## 5. ロジック実装用タグ
*   `Total_Units >= 124` (卒業条件)
*   `Foundation_Units >= 36`
*   `Professional_Units >= 36`
*   `Other_Graduation_Units >= 52`
*   `Foundation_Overflow_Countable <= 10`
*   `Promote_Condition`: `(GraduationRequirementUnits >= 50) AND ((Cumulative_GPA >= 1.0) OR (Year2_GPA >= 1.0))`
*   `Semester_CAP = 24`
*   `Teacher_Training`: `Section 1+2+3` 内の特定科目 + `専用科目表8,9` の全修得
*   `Other_Dept_Limit`: `FB学科科目 <= 12`
*   `Other_University_Limit <= 30`
*   `Upper_Grade_Courses_Disallowed = TRUE`
*   `Certification_Recognition`: `ITパスポート` / `基本情報技術` は申出により単位認定可

## 6.5. 主要な直列前提関係メモ
判定ロジックへ反映する主要な順序関係を、系列ごとに整理する。

- 基礎教育:
  プラクティカル・イングリッシュⅠ → Ⅱ → Ⅲ → Ⅳ、フレッシュマンセミナーⅠ → Ⅱ → 基礎演習Ⅰ → 基礎演習Ⅱ

時間割PDFで `日本語力Ⅰ`・`日本語力Ⅱ` はともに2年前期、`数的処理Ⅰ`・`数的処理Ⅱ` はともに1年後期の開講実績が確認できたため、この2系列については本プロジェクトでは `I` を `II` の前提科目として扱わない。
- コース共通:
  情報学概論 → ＩＴマネジメント → ＩＴストラテジ → 基本情報技術、情報メディア論 → 情報社会の倫理と職業、基礎演習Ⅱ → 専門演習Ⅰ → 専門演習Ⅱ → 卒業演習Ⅰ → 卒業演習Ⅱ
- 情報システム:
  プログラム演習Ⅰ → プログラム演習Ⅱ → プログラム演習Ⅲ → プログラム演習Ⅳ、アルゴリズムとデータ構造 → ゲームプログラミング、コンピュータネットワークⅠ → Ⅱ → モバイル情報論、情報システム入門 → 情報基礎理論 → メディア情報技術 → システム開発論 → データベース
- 映像メディア:
  映像制作演習Ⅰ → 映像制作演習Ⅱ → 映像加工技術 → メディアアート、デジタルアート入門 → アニメーション
- サウンド制作:
  デジタルサウンド入門 → 音響実務、ＭＩＤＩ制作演習Ⅰ → Ⅱ → 音楽理論 → サウンドクリエーション
- メディアデザイン:
  Ｗｅｂプログラミング → Ｗｅｂデザイン → Ｗｅｂ解析

共有科目として複数系列に現れる `ＣＧ基礎`、`マルチメディア`、`ドキュメンタリー演習`、`放送・配信論` などは、表示上の接続と履修判定上の前提を分けて扱う。

## 7. 教職課程の履修順メモ
教職課程に関する科目は、単一科目どうしの前提関係というより、半期ごとの必修群を順に履修する構造として扱う。

| 履修時期 | 必修群 |
| :--- | :--- |
| 1年前 | 教育原理、教師論 |
| 1年後 | 教育課程論、教育の制度と経営 |
| 2年前 | 教育心理学、特別活動論、生徒進路指導論 |
| 2年後 | 教育相談論、特別支援教育、総合的な学習の時間の指導法 |
| 3年前 | 教育の方法と技術（情報通信技術の活用含む）、情報科教育法Ⅰ |
| 3年後 | 教育実習指導、情報科教育法Ⅱ |
| 4年前 | 教育実習 |
| 4年後 | 教職実践演習（高校） |

教職課程の基礎資格として、`日本国憲法` 2単位、`スポーツⅠ`・`スポーツⅡ` 合計2単位、`プラクティカル・イングリッシュⅠ` 2単位、`情報リテラシー` 2単位も必要である。
