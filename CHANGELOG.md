# Changelog

## 1.05.0 - 2026年度版 Ver.1.05 音響・演出大改修版

### Added

- 演出メニューに `アニメ` ON/OFF を追加し、SE/BGMとは独立して画面アニメーションを停止できるようにした
- BGMロールパネルに `Arrange` Seed、`Melody` Seed、BPM調整を追加
- `Arrange` Seed と `Melody` Seed に、それぞれランダム値を設定するボタンを追加
- 卒業要件、基礎教育、専門教育、その他の進捗バーに増減アニメーションを追加
- BGM/SE/アニメーション調整の設計メモを `docs/audio-and-effects.md` に追加

### Changed

- BGM生成をWeb Audio時刻で先読み予約する方式へ変更し、描画更新やUI操作による途切れを抑制
- BGM用音源ノードを追跡し、BPM変更時には予約済みBGMを破棄して現在BPMで即時再スケジュールするように変更
- SEとBGMの出力経路を分離し、SE側だけを軽いリミッターへ通す構成に変更
- BGMのアレンジ用Seedと基本メロディ用Seedを分離し、同じ履修状態でもアレンジとメロディを別々に試せるように変更
- メニュー開閉時の処理を軽量化し、ヘッダー系メニューや絞込パネルの開閉で全体再描画を避けるように変更
- レイヤー表示への切り替え時に、科目群パネルが左から順にスライドインする演出を追加

### Fixed

- SEをOFFにしてもボタン操作時にBGMが一瞬止まる問題を修正
- `pointerdown` イベントオブジェクトがBGM再起動フラグとして扱われる問題を修正
- BGMのBPM変更が、Melody Seed変更時まで実音へ反映されにくい問題を修正
- BGMステップ計画キャッシュにBPMが含まれず、古いテンポの `nextDelay` が再利用される問題を修正
- レイヤー表示切り替え演出で、各パネルが同時に表示されてしまう問題を修正
- アニメOFF時にパーティクルや画面揺れが残る問題を抑制

## 1.04.0 - 2026年度版 Ver.1.04 大改修版

### Added

- 科目表示モードに `レイヤー` を追加
- 科目グループを地下層、基礎層、コース共通層、各コース層、教職課程層として重ねるクオータービュー風表示を追加
- レイヤー選択時に、遮蔽する手前レイヤーを透過させ、選択レイヤーを読みやすくする表示制御を追加
- レイヤー表示でも通常ツリーと同じ科目間リンクを描画し、アクティブでないレイヤーのリンクは非表示にする制御を追加
- レイヤー表示の設計と調整履歴を `docs/layer-view-mode.md` に追加

### Changed

- 表示切り替えUIを `一覧`、`ツリー`、`レイヤー` の単一選択ボタンに変更
- レイヤー表示の各パネル、科目セル、履修時期メニューを同じスキュー方向へ揃え、クオータービューとして整合するように調整
- レイヤーパネルの影を削除し、重なり量、座布団サイズ、左寄せ、スクロールバー抑制を調整
- レイヤーのコース重なり順をコース選択メニューと同じ `情報システム`、`サウンド制作`、`メディアデザイン`、`映像メディア` に統一
- ツリー表示とレイヤー表示で、履修時期を選択した科目を選択時期の列へ移動し、未配置に戻すと標準位置へ戻すように変更
- メディアデザインコースの推奨科目 `デジタルアート入門` が映像メディア色にならないよう、推奨表示色をメディアデザイン色へ修正

### Fixed

- レイヤー表示初期状態で科目セルのスキューが乱れる問題を修正
- レイヤー下層をアクティブにした時に文字の透明度が下がったままに見える問題を修正
- 履修時期選択メニューが隣列や他科目の下に隠れて操作しにくい問題を調整
- レイヤー全体のパネルサイズが内部科目パネルより不足し、下にはみ出る問題を修正

## 1.0.2 - 2026年度版 Ver.1.02

### Changed

- 科目一覧のリスト表示を、基礎科目・専門共通科目・各コース・教職課程・他学科履修のグループ単位で多段化
- 画面横幅に応じて科目一覧の列数が増えるように調整
- コース選択メニューの並びをツリー表示のコース順に統一
- コース選択メニューと専門科目サマリーの配色を、ツリー表示のコース色に合わせて統一

## 1.0.1 - 2026年度版 Ver.1.01

### Changed

- release版の公開タイトルを `超履修計画2026` に変更
- BGMの頭出しをコース変更時のみに限定
- 科目追加、科目削除、全必修、全削除、教職ON/OFFでは、BGMを止めずに履修状態のアレンジを継続反映
- BGM解析結果を履修状態ごとにキャッシュし、毎ステップの再計算を削減
- seed付きノイズ音のAudioBufferをキャッシュし、BGM/SE演出の内容を変えずに音声処理負荷を削減
- BGMの長時間ループ中にAudioNodeが残り続ける問題を避けるため、停止後の明示的なdisconnectとtick失敗時の継続ガードを追加

## 1.0.0 - 2026年度版 Ver.1.0

### Added

- カリキュラムツリー表示モードを追加
- ノードによる科目配置、科目間接続、コース別配置の可視化
- ツリー上の科目ノードから履修時期を選択して履修計画へ追加する操作
- 基礎教育科目、集中講義、資格取得、他学科、教職課程を含むツリー表示
- コース必修、教職必修、履修推奨科目の枠線・ホバー表示
- 教職課程科目の半期必修群間を全結合で表示する接続
- 卒業要件、基礎教育、専門教育の進捗バーを内訳つき表示に変更
- 専門科目全体のコース別内訳を色分け表示
- コース選択、絞り込み、GPA条件、演出設定を展開式メニュー化
- `登録` 行に `全必修` と `全削除` を配置
- コース別BGM、SE、履修状態によるBGMアレンジを追加
- BGMの変化を確認するピアノロール風可視化パネルを追加
- 登録、解除、絞り込み、全必修、全削除時の逐次アニメーションを追加
- Ver.1.0の仕様・演出意図まとめを `docs/ver-1.0-overview.md` に追加

### Changed

- コース選択操作を `コース変更＋必修追加` と `コース変更＋他コース解除` に整理
- ツリー表示のスマートフォン表示では、科目名が読める横幅を優先
- SE/BGMの出力段にフィルターとリミッターを追加し、発音の立ち上がりと終端を調整
- 一時生成物の `tmp/` をGit管理対象外に整理
