# Ver.1.04 レイヤー表示メモ

2026年度版 `Ver.1.04` で追加した、科目グループごとのレイヤー表示に関する設計・調整メモ。

## 目的

- 通常ツリーとは別に、科目グループの階層関係を立体的に把握できる表示モードを用意する。
- 基礎教育、コース共通、各コース、専門選択、他学科、教職課程の位置づけを、重なったレイヤーとして見せる。
- 通常ツリーの履修登録、履修時期変更、科目間リンク、科目名フィット処理をできるだけ共通利用する。

## 表示モード

- 表示切り替えは `一覧`、`ツリー`、`レイヤー` の単一選択ボタンで行う。
- レイヤー表示は `state.viewMode === "layer"` で分岐する。
- 通常ツリーとレイヤー表示は、共通の `visibleTreeNodes()` と `buildTreeNodeElement()` を使う。

## レイヤー順

レイヤーの深度は `layerDepthForSection()` で管理する。

| 深度 | セクション |
| --- | --- |
| -1 | 他学科履修 |
| 0 | 基礎教育科目 / 基礎教育必修 |
| 1 | 基礎教育選択 |
| 2 | コース共通 |
| 3 | 情報システムコース |
| 4 | サウンド制作コース |
| 5 | メディアデザインコース |
| 6 | 映像メディアコース |
| 7 | 専門選択 |
| 8 | 教職課程に関する科目 |

コース層の順序は、コース選択メニューの順序と合わせる。

## レイヤー選択

- レイヤーパネルをクリックすると、そのレイヤーをアクティブにする。
- アクティブなレイヤーを再クリックすると選択解除する。
- アクティブレイヤーを遮蔽する手前のレイヤーは透過させる。
- 透過状態のレイヤーをクリックした場合も、そのレイヤーへ直ちにアクティブ状態を切り替える。
- 下層レイヤーがアクティブな時にアルファ値が下がったままに見えないよう、アクティブレイヤーと開いている科目メニューは前面に出す。

## 見た目の方針

- レイヤーパネルや科目セルは、右上だけを持ち上げる変形ではなく、平行四辺形としてクオータービュー風に見せる。
- 立体感のためのドロップシャドウは使わない。
- レイヤーは各レイヤーの見出し位置を基準に重ね、見出し以外の領域が大きく覆い隠されるようにする。
- 開講期ラベルは、重なりの中で不要な場合は目立たせず、科目セルの配置を優先する。
- レイヤー全体の座布団は、内部の科目パネルが下へはみ出さない高さに動的調整する。
- 内部スクロールバーは出さず、レイヤー全体の高さを科目パネルに合わせる。

## 科目セル

- レイヤー表示の科目セルは、通常ツリーと同じノード生成ロジックを使う。
- 科目名はセル内で上下中央に配置する。
- 補足表示がOFFの場合、系列ラベルと補足行は表示しない。
- メディアデザインコースの推奨科目 `デジタルアート入門` は、映像メディア色ではなくメディアデザイン色で表示する。
- Safariやブラウザ表示倍率の違いでも科目名が見切れにくいよう、通常ツリーと同じ科目名フィット処理を使う。

## 履修時期メニュー

- レイヤー表示でも、通常ツリーと同じ履修時期選択メニューを開く。
- メニューは科目や隣列の下に隠れないよう、開いているノード、列、レイヤーを前面に出す。
- メニューのスキュー方向は科目セルと揃える。
- `未配置` を選ぶと履修登録を削除する。

## 履修時期による配置移動

- ツリー表示とレイヤー表示では、履修登録済みの科目を `plannedTerm(course)` の列へ表示する。
- 履修を削除した科目は、公式ノードや科目データが持つ標準開講期へ戻る。
- 表示フィルターは元の科目データを基準にするため、履修時期を移動しても意図せずフィルター外へ消えない。

## 科目間リンク

- レイヤー表示でも通常ツリーと同じ `visibleTreeEdges()` を使って科目間リンクを描画する。
- レイヤーが未選択の場合、リンクは表示しない。
- レイヤーがアクティブな場合、そのレイヤーの科目に接続するリンクだけを表示する。
- これにより、非アクティブな科目パネルのリンクが画面を覆って読みにくくなることを避ける。
