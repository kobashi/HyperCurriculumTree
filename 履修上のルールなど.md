# 情報メディア学科 履修・卒業判定ルール詳細資料

この資料は、名古屋文理大学 情報メディア学科の履修計画・卒業判定ツールで参照するルールを、2023年度以降入学生向けに整理したものです。

---

## 1. 卒業要件の基本構造

卒業には、卒業要件に算入される科目で合計124単位以上の修得が必要です。

| 区分 | 必要単位 | 算入ルール |
| :--- | :---: | :--- |
| 基礎教育科目 | 36以上 | 必修22単位 + 選択14単位 |
| 専門教育科目 共通必修 | 20 | 学科共通必修10科目 |
| 専門教育科目 コース必修 | 16以上 | 所属コース指定の必修8科目 |
| 卒業に必要なその他の科目 | 52以上 | 基礎選択超過の振替、専門選択、他学科履修、資格認定など |
| 合計 | 124以上 | 卒業要件外単位は含めない |

---

## 2. 単位算入・振替ルール

### 2.1. 基礎教育科目 選択枠の超過振替

基礎教育科目の選択区分で14単位を超えて修得した場合、以下の通り処理します。

| 対象 | 算入先 | 上限 |
| :--- | :--- | :---: |
| 基礎選択14単位まで | 基礎教育科目 | 14 |
| 14単位を超えた分 | 卒業に必要なその他の科目 | 最大10 |
| 超過分の11単位目以降 | 卒業要件外 | 上限なし |

### 2.2. 集中講義・資格取得による単位認定

2023年度以降入学生は、下記科目について集中講義としての履修、または指定資格の取得により単位認定を受けられます。

| 科目名 | 認定単位 | 取得方法 | 算入区分 | 卒業要件算入 | 履修上限 |
| :--- | :---: | :--- | :--- | :---: | :---: |
| ITパスポート | 2 | 集中講義 / 資格取得 | 卒業に必要なその他の科目 内の専門選択 | TRUE | 対象外 |
| 基本情報技術 | 2 | 集中講義 / 資格取得 | 卒業に必要なその他の科目 内の専門選択 | TRUE | 対象外 |

集中講義として履修した場合も、資格取得により単位認定を受けた場合も、学期ごとの履修上限には含めません。同一科目について、集中講義による単位認定と資格取得による単位認定を重複して算入することはできません。

### 2.3. 他学科履修

フードビジネス学科の専門教育科目は、卒業までに12単位まで卒業要件に算入できます。

| 対象 | 算入先 | 上限 |
| :--- | :--- | :---: |
| フードビジネス学科科目 | 卒業に必要なその他の科目 内の専門選択 | 12 |
| 12単位を超えた分 | 卒業要件外 | 上限なし |

---

## 3. 卒業要件外単位

以下は卒業要件124単位に算入しません。

- 学科特別科目
- 他学科履修の12単位超過分
- 基礎教育科目の選択14単位を超えた分のうち、その他枠へ振替可能な10単位をさらに超えた分
- 教職課程専用科目のうち、卒業要件の専門選択に合致しない科目

---

## 4. 進級要件

2年次から3年次への進級時に、2年次後期終了時点で以下を満たす必要があります。

- 卒業要件科目50単位以上の修得
- 通算GPA 1.0以上、または2年次の年間GPA 1.0以上

1年次から2年次、および3年次から4年次への進級要件はなく、自動的に進級します。

---

## 5. 履修上限

### 5.1. 基本上限

- 1学期につき24単位まで
- 年間GPAが3.7以上の場合、次年度は各学期26単位まで

### 5.2. 履修上限に含めない単位

以下は履修上限の対象外です。

- 海外事情
- アウトドアスポーツI・II
- インターンシップ
- ボランティア活動
- 情報メディア特別演習I・II
- 教職課程に関する科目
- その他集中講義科目
- 集中講義として開講されるITパスポート・基本情報技術
- 資格取得による単位認定
- 学科特別科目

上記のうち、基礎教育科目として開講時期が定められている集中講義は、その開講期に配置します。例: 海外事情は前期、アウトドアスポーツIは前期、アウトドアスポーツIIは後期、ボランティア活動は後期。

---

## 6. 履修制限

- 上級年次科目は、自身の学年より上の年次では履修できません。
- すでに単位を修得した科目を再履修することはできません。
- 第2外国語IIは、原則として同一言語のIの修得後に履修します。
- 基礎演習I・IIは、原則として同一教員のもとで連続履修します。
- 専門演習と卒業演習は、原則として同一教員のもとで継続履修します。

---

## 7. 教職課程

4年次に教育実習を行うには、3年次終了時点で以下を満たす必要があります。

- 免許状取得に必要な基礎教育科目8単位を修得済み
- 教科に関する科目34単位以上を修得済み
- 教育基礎科目等について、実習・実践演習を除く必要な21単位を修得済み

---

## 8. 実装用チェック項目

| チェック | ロジック |
| :--- | :--- |
| 卒業要件単位 | `GraduationUnits >= 124` |
| 基礎教育 | `BasicRequired >= 22` and `BasicElectiveCounted >= 14` |
| 基礎選択振替 | `Transfer = min(max(BasicElectiveRaw - 14, 0), 10)` |
| 基礎選択要件外 | `Outside = max(BasicElectiveRaw - 24, 0)` |
| 他学科算入 | `OtherDeptCounted = min(OtherDeptRaw, 12)` |
| 他学科要件外 | `OtherDeptOutside = max(OtherDeptRaw - 12, 0)` |
| 集中講義・資格認定 | `RecognitionUnits` をその他52単位枠へ算入し、履修上限から除外。同一科目の集中講義と資格取得は重複不可 |
| 進級 | `UnitsByEndOfSecondYear >= 50` and `(CumulativeGPA >= 1.0 or SecondYearGPA >= 1.0)` |
| 履修上限 | `LimitCountedUnits <= 24`、GPA3.7以上の場合は `<= 26` |
